## Project Description
This project demonstrates Unity fundamentals including:
- Scene creation
- Parent-child hierarchy
- Primitive 3D object modeling
- Materials and textures
- Physics interaction
- Movement and camera follow scripts

## Scene Features
- Grassy ground
- Walking track
- Human structure made using Unity primitives
- Camera follow system
- Player movement

## Tools Used

- C#
- Built-in Unity primitives
